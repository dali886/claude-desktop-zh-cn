怎么用

核心就四步。Windows 必须管理员；Mac 如提示未识别开发者，在隐私设置里允许打开。

退出 Claude

托盘/程序坞彻底退出，任务管理器里不要残留进程。

运行补丁

Windows 右键管理员运行；Mac 双击「一键汉化-Mac.command」。

等待完成

自动定位安装目录、写入 zh-CN、补丁语言白名单。

重新打开

系统为简体中文时一般自动生效；否则到设置里选中文。

Claude 更新后可能换新目录，界面变回英文是正常现象——再运行一次补丁即可。

常见问题

使用前请阅读：这是非官方社区工具。

提示没有权限写入怎么办？

Windows：必须右键 exe → 以管理员身份运行，UAC 点「是」；并确认 Claude 已完全退出。可暂时关闭杀毒实时保护后再试。公司电脑若锁定 WindowsApps，需要管理员策略放行。

会覆盖我原来的英文界面吗？

不会。工具只新增 zh-CN 文件并补丁白名单，不改 en-US / ja-JP 等现有语言包。

Mac 双击没反应 / 无法打开？

系统设置 → 隐私与安全性 → 仍要打开。或在终端进入解压目录执行：
xattr -dr com.apple.quarantine . && chmod +x 一键汉化-Mac.command && ./一键汉化-Mac.command
